## 프로젝트 개요
- **운영 환경**: Windows Server + Apache Tomcat 8.5 + Oracle 11g
- **기간**: 20XX년 ~ 현재 (운영 중)
- **주요 기능**:
  - 근태 관리 (출퇴근, 연차, 초과근무)
  - 급여 계산 (기본급, 수당, 공제, 4대보험)
  - 직원 관리 (입사, 퇴사, 인사이동)
  - 보고서 출력 (PDF, Excel)
